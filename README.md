# CTF & Penetration Testing Writeups

A personal collection of writeups documenting Capture The Flag (CTF) challenges and penetration testing lab exercises completed across various security training platforms. Each report covers a distinct scenario, detailing the methodology, tools, commands, and reasoning applied at every stage of the assessment.

---

## About This Repository

These writeups serve as a personal knowledge base and portfolio of practical offensive security skills developed through hands-on lab environments. They are written in the style of formal technical reports — mirroring the documentation standards expected in real-world penetration testing engagements — and span a range of attack techniques across network services, web applications, Windows and Linux systems, database servers, and post-exploitation scenarios.

Each report includes:

- **Reconnaissance and service enumeration** methodology
- **Step-by-step exploitation walkthroughs** with commands and tool output
- **Analysis and reasoning** behind each technique and decision
- **Vulnerability summaries** with risk classifications
- **Security recommendations** mapped to each finding

Reports are organised by platform and module. New writeups are added progressively as labs are completed.

## Flag Disclaimer

> **All flag values presented in these writeups have been redacted and replaced with placeholder strings (e.g. `FLAG{######################################}`).**

This is done deliberately and out of respect for the platforms hosting these labs and the integrity of their learning environments. Publishing actual flag values would undermine the experience of other students working through the same exercises and may conflict with the terms of service of the respective platforms.

The redaction does not affect the educational value of the writeups: the methodology, commands, tools, and reasoning documented in each report are complete and sufficient to understand and reproduce the techniques independently in an authorised environment.
